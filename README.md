# 🎓 高考志愿填报管理表 · Gaokao Volunteer Planner

> 一个**单文件、零依赖、可离线**的高考志愿填报规划工具。双击即用，数据只存在你自己的浏览器里。
新高考"院校专业组"模式下，志愿填报常常要在一张大表里反复权衡：冲哪些、稳哪些、保哪些，分散在哪些地区，选科对不对得上，顺序怎么排……
这个工具把这些都收进**一个 HTML 文件**里——不用联网、不用注册、不用安装，所有数据保存在本机浏览器，关掉再开依然在。

---

## ✨ 亮点一览

| 功能 | 说明 |
| --- | --- |
| 🗂️ **多方案分支管理** | 像 Git 分支一样维护多套志愿表（冲名校版 / 保稳妥版…），可新建、**复制(fork)**、重命名、删除，标签一键切换对比 |
| 🏫 **院校 → 专业组 两级结构** | 一所院校下挂多个专业组，可展开/折叠；弹窗查看"同一院校的专业组数量"与明细 |
| 🎯 **位次自动判梯度 + 风险灯** | 填入你的位次，每个专业组按往年位次自动亮灯 🟢稳妥 / 🔵较稳 / 🟠可冲 / 🔴偏险，一键回填冲稳保 |
| 📊 **数量 / 比例进度预警** | "已填 X / 上限"进度条；没有保底、冲占比过高、超出上限时主动提醒 |
| ✅ **选科匹配校验** | 填入你的选科，要求对不上的专业组自动标红并汇总数量（识别 `物理+化学`、`物理或历史`、`不限` 等） |
| 🗺️ **填报地区可视化** | 中国省级行政区示意地图，按各省专业组数量热力着色，点击省份即可筛选 |
| ↕️ **拖拽排序** | 拖动手柄给院校、或同院校内的专业组排序（平行志愿按顺序投档） |
| 🔗 **学校官网管理** | 每所院校记录官网网址，一键在新标签打开 |
| 📝 **生成填报清单** | 按顺序导出"院校代码 + 专业组代码 + 专业"清单，支持**打印 / 复制 / 导出 CSV**，照着往官方系统誊抄 |
| 🌗 **现代极简 UI + 夜间模式** | 无边框留白设计、紫罗兰点缀色、字重分层；一键切换日间/夜间并自动记忆 |
| 💾 **本地保存 + 备份迁移** | 自动存 `localStorage`；可导出/导入 JSON 全量备份，导出当前方案为 CSV(Excel 可读) |

## 🚀 快速开始

无需任何环境，三步即可：

1. 下载仓库里的 **`高考志愿填报管理表.html`**（或 `Code → Download ZIP`）。
2. **双击**该文件，用任意现代浏览器（Chrome / Edge / Firefox / Safari）打开。
3. 直接在页面上填写、规划即可——所有改动自动保存。

> 💡 也可以把它放进网盘/U 盘随身带，或托管到任意静态站点（GitHub Pages 等）；它是纯前端页面，不需要后端服务。

```bash
# 可选：克隆到本地
git clone https://github.com/miaomaioji/-.git
# 然后双击打开 高考志愿填报管理表.html
```

## 🧭 使用指南

- **顶部「我的定位与填报进度」**：填入位次 / 分数 / 选科 / 可填上限 → 下方表格自动给出风险灯与选科校验，点 `⚡ 按位次自动判梯度` 一键回填冲稳保。
- **蓝/灰色行 = 院校，缩进行 = 专业组**：院校行最左 `▾/▸` 展开折叠；`＋专业组` 给同校添加组；`📋 详情` 查看该校专业组数量。
- **省份地图**：点击任一省份方块，只看该地区的院校；再次点击或点 `✕` 取消筛选。
- **拖拽手柄 `⠿`**：拖动给院校或同院校内的专业组排序。
- **顶部工具**：`📝 生成填报清单`（打印/复制/导出）、`📊 组数统计`、`⬇ 导出备份(JSON)` / `⬆ 导入备份`、`🌙/☀️` 切换主题。
- 页面内有 **`? 帮助`** 弹窗，随时查看完整说明。

## 🔒 数据与隐私

- **完全本地**：所有数据保存在你浏览器的 `localStorage`，**不上传任何服务器**，无埋点、无网络请求。
- **离线可用**：首次打开后断网也能正常使用。
- **备份建议**：清理浏览器数据 / 更换设备前，请用 `导出备份(JSON)` 保存，到新环境用 `导入备份` 恢复。

## 🛠️ 技术细节

- 纯 **HTML + 原生 CSS + 原生 JavaScript**，单文件、无框架、无构建步骤、无第三方依赖。
- 中国地图为 **CSS 方块示意图**（schematic tile-map），离线零依赖，强调方位与分布而非精确边界。
- 主题系统基于 CSS 变量 + `data-theme`，切换无白屏闪烁，首次访问跟随系统深色偏好。
- 兼容近期版本的 Chrome / Edge / Firefox / Safari。

## ❓ 常见问题

<details>
<summary><b>数据会被上传吗？</b></summary>
不会。工具是纯前端页面，数据仅存于本机浏览器，没有任何联网上传行为。
</details>

<details>
<summary><b>换电脑或清理浏览器后数据会丢吗？</b></summary>
会。请先用「导出备份(JSON)」保存文件，到新环境用「导入备份」恢复。
</details>

<details>
<summary><b>里面的位次/分数准确吗？</b></summary>
预置内容均为<b>示例演示数据</b>，请以你所在省份教育考试院公布的官方招生计划、历年投档线/位次为准。
</details>

<details>
<summary><b>地图是精确地图吗？</b></summary>
不是，是方块<b>示意图</b>，用于直观查看地区分布，胜在离线、零依赖、可点击筛选。
</details>

<details>
<summary><b>手机能用吗？</b></summary>
可以打开使用；宽表格在小屏需横向滑动，移动端卡片化布局已列入路线图。
</details>
